# YouTube 영상 다운로더

## 소개
YouTube 영상 URL을 입력하면 영상을 다운로드하여 .exe 실행 파일로 만들어주는 데스크톱 애플리케이션입니다.

## 주요 기능
- **URL 입력**: YouTube 영상 URL을 붙여넣기하여 다운로드
- **화질 선택**: 360p, 720p, 1080p, 4K 등 원하는 화질 선택
- **포맷 선택**: MP4, WEBM, MP3(오디오만) 등 다양한 포맷 지원
- **배치 다운로드**: 여러 영상 URL을 한 번에 다운로드
- **진행률 표시**: 다운로드 진행 상황을 실시간으로 확인
- **자동 .exe 변환**: 다운로드 완료 후 실행 파일로 자동 생성

## 사용 방법
1. YouTube 영상 URL 복사
2. 프로그램에 URL 붙여넣기
3. 화질 및 포맷 선택
4. 다운로드 버튼 클릭
5. 완료된 파일 확인

## 기술 스택
- **Python**: 핵심 로직 및 영상 처리
- **yt-dlp**: YouTube 영상 다운로드 라이브러리
- **PyInstaller**: .exe 파일 변환
- **Tkinter/PyQt**: GUI 인터페이스

## 시스템 요구사항
- Windows 10 이상
- Python 3.8 이상 (빌드 환경)
- 인터넷 연결

## 주의사항
- 저작권이 있는 영상은 개인 소장 목적으로만 사용하세요
- YouTube 이용약관을 준수하세요

## 빌드 방법

### 1. Python 및 패키지 설치
```bash
pip install yt-dlp pyinstaller
```

### 2. .exe 파일 빌드 (PyInstaller)
```bash
pyinstaller --onefile --windowed --name "YouTubeDownloader" youtube_downloader.py
```
`dist\YouTubeDownloader.exe`가 생성됩니다.

### 3. Windows 설치 프로그램 빌드 (Inno Setup)

#### Inno Setup 설치
https://jrsoftware.org/isdl.php 에서 Inno Setup을 다운로드하여 설치합니다.

#### 설치 프로그램 컴파일
```bash
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" setup.iss
```
또는 `setup.iss` 파일을 Inno Setup에서 열어서 Compile(F9)합니다.

#### 결과물
`installer_output\YouTubeDownloader_Setup.exe`가 생성됩니다.
이 설치 프로그램을 실행하면 YouTube Downloader가 자동으로 설치됩니다.

### 자동 빌드
`build_installer.bat`을 더블클릭하면 PyInstaller 빌드까지 자동으로 진행됩니다.

### FFmpeg 설치 (선택사항)
고화질 다운로드 및 오디오 변환을 위해 FFmpeg 설치를 권장합니다.
https://ffmpeg.org/download.html
